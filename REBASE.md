# limpieza de commits

## pasos realizados
1. realice varios commits con mensajes pocos claros.
2. use git rebase -i HEAD~3 para editar el historial.
3. cambie los mensajes usando reword.
4. fusione commits con squash.
5. actualize el repositorio con git push --force

## decisiones
- fusione commits porque eran cambios relacionados.
- mejore los mensajes para que sean mas descriptivos.
